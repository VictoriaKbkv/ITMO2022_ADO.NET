Приложение позволяет просматривать данные и вносить изменения в таблицу [ShareDetails].[Shares] базы данных ApressFinancial.
Приложение разработано с использованием технологий Entity Framework и Windows Forms (использованы формы, разработанные в рамках зачетного задания
по курсу "Разработка Windows-приложений на Csharp").

При запуске приложения открывается диалоговое окно для авторизации.
Login: User1
Password: Qwerty123
Ппри авторизации используется текущее имя пользователя и пароль, под которым пользователь авторизовался в системе (Integrated Security=SSPI).

Основной элемент управления – ToolStrip с кнопками: Show ALL, Delete, Save, Discard Changes.

Область ввода/вывода данных – DataGridView. 

В БД в таблице [ShareDetails].[Shares] 4 столбца. 
Приложение выводит на экран и позволяет задавать значения при вводе данных только для 3 столбцов: ShareDesc, ShareTickerID, CurrentPrice. 
Значения столбца ShareID (ключевое поле) в соответствии с настойками БД задаются автоматически. 
Предполагается, что эта информация не нужна пользователю и используется только для работы БД (ключевое поле, ограничение внешнего ключа для других таблиц).
При сохранении новых строк через приложении в БД ShareID генерируется автоматически.

1. Вывод данных из БД на экран: кнопка Show All.

2. Изменение данных в таблице: пользователь может редактировать данные в области GridView. 
Если нажать кнопку Save данные сохраняются в БД. 
Если нажать кнопку Discard Changes все внесенные изменения отклоняются и не сохраняются в БД.
Для удобство пользователя строки, которые были отрадактированы, подсвечиваются желтым.

3. Добавление новых данных: пользователь может добавить новые строк в области GridView.
Если нажать кнопку Save новые строки сохраняются в БД. 
Если нажать кнопку Discard Changes все внесенные изменения отклоняются и не сохраняются в БД.
Для удобство пользователя новые строки подсвечиваются желтым.
В StatusStrip ведется подсчет добавленных строк.

4. Удаление данных: пользователь может выделить одну или несколько строк в области GridView и нажать кнопку Delete.
Строки будут удалены из области GridView. В StatusStrip ведется подсчет удаленных строк.
Для сохранения изменений (удаления данных) в БД необходимо нажать кнопку Save.
Чтобы отменить удаление нужно нажать кнопку Discard Changes.
